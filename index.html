    Semua maklumat kewangan anda disimpan secara eksklusif dalam simpanan pelayar tempatan anda (Local Storage) untuk keselamatan privasi penuh.

    Data Simulasi Bank

    Anda boleh memuatkan semula data simulasi asal dari **ABC Bank Penyata Akaun** (Ahmad Bin Abdullah) pada bila-bila masa. Ini akan menimpa sebarang rekod tambahan yang telah anda masukkan secara manual.

    Konfigurasi Firebase Database

    Hubungkan aplikasi ini ke akaun **Firebase Cloud Firestore** anda sendiri untuk sandaran dan penyegerakan data awan secara masa nyata.
